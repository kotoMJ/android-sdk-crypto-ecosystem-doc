# Introduction

Tracker SDK offert interfaces pro captandis Lirum track fundatis in Tracker API 2.0.

## API Feature list

### Basis notitia

- Get all competitions
- Get all seasons
- Get all clubs in a season, optionally filter by competition
- Get single club detail

### Ceramina

- Accipe certamen cedulas in specifica periodo optione spargere per competitiones et fustes
- Observa certamen schedula mutationes in vivis certaminibus
- Accipe certamen detail cum available ambitibus 
  - arbitri 
  - lusores 
  - lusores statistica 
  - fustes statistica 
  - priora certamina inter duas fustes 
  - clavis eventus in vivo certamine 
  - commentarius eventus in vivo certamine 
- Observa certamen detail mutationes in vivo certamine

## Classificationes

- Accipe Classificationes in tempore et competitione, optione spargere per categoriam (divisio, conferentia,
  foederaPoculum) et typus
- Observa classificationes mutationes ad reflectere positiones in vivis certaminibus
